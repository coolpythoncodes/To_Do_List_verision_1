# To_Do_List_verision_1
A To-Do List app built with Django and Bootstrap.

Features of the To-Do List webapp

-1 Clear all task at once in To-Do List

-2 Remove one task at a time

-3 Add task

## Project setup
```
python manage.py runserver2
```
